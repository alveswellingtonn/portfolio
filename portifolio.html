Projeto

Sistema de automação residencial

Esse trabalho foi desenvolvido com um sistema de automação residencial para os casos de usuário que tenha alguma deficiência para realizar as atividades de acender e apagar lâmpada e abrir e fechar portas e portão, utilizando um aplicativo capaz de controlar lâmpada através de um celular onde a comunicação é feita via Bluetooth capaz de receber uma informação de um usuário, usando também um sensor de sinais onde travas de portão são acionadas e abertas somente quando encostar-se ao leitor, isso através de uma programação já pré estabelecida que libere o acesso, onde será mostrada uma mensagem no LCD se foi permitida ou negada à liberação.

Tecnologia

Placa Arduino UNO R3, LCD 16x2 1602A, Módulo RFID - RC522, Módulo I2C, Cartão do RFID – RC522, Módulo Bluetooth HC-05, LED’s de alto brilho, Cabo USBAxB, Resistores, IDE Arduino versão 1.8.1, Aplicativo Arduino Bluetooth Control Device versão 1.72. Para a programação do sistema foi utilizada a Linguagem C

Aplicativo para calcular o IMC
